# `Contribute`

Contributors are welcome! If you want to contribute to the documentation, you can follow the steps below.

<Callout emoji="ðŸ”°">Active contributors will get rewards!</Callout>

### Make a Github Account

If you don't have a Github account, you can make one [here](https://github.com/signup).

### Fork the Repository

Fork the repository by clicking the fork button on the top right of the page. This will create a copy of the repository in your account.

[![Fork Button](https://imgur.com/afZ3OfO.png)](https://imgur.com/afZ3OfO.png)

[![Fork Button](https://imgur.com/tZbWBrb.png)](https://imgur.com/tZbWBrb.png)

### Download the git client and git

Download the git client [here](https://cli.github.com/).
Download git [here](https://git-scm.com/downloads).

### Download the repository with cli

Open the git client and type the following command:
```
gh repo clone <your github username>/jolyne-documentation
```
[![Git Client](https://imgur.com/DySXUt3.png)](https://imgur.com/DySXUt3.png)

### Install Visual Studio Code

You can download Visual Studio Code [here](https://code.visualstudio.com/).

### Open docs folder in Visual Studio Code

Once you installed and launched Visual Studio Code, open the folder from the repository you just downloaded.

[![Open Folder](https://imgur.com/o1HXMBr.png)](https://imgur.com/o1HXMBr.png)

### Install extensions

Go on the 5th icon on the sidebar (from the top) and click on the extensions tab.
    
    [![Extensions](https://imgur.com/0jMkvnu.png)](https://imgur.com/0jMkvnu.png)

Install the following extensions:

- [Auto Rename Tag](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)
- [Markdown Preview Enhanced](https://marketplace.visualstudio.com/items?itemName=shd101wyy.markdown-preview-enhanced)

### Edit the files

Go on an mdx file and use ***Shift + Ctrl (CMD on Mac) + P*** and write **">markdown"**, then select **"markdown preview enhanced: open preview"**

[![Markdown Preview](https://imgur.com/ZQW2vQl.png)](https://imgur.com/ZQW2vQl.png)

Now, you're ready to edit the files and should see the file's preview, if you don't know how to use markdown, you can learn it [here](https://www.youtube.com/watch?v=34_dRW42kYI).

[![Markdown Preview](https://imgur.com/NuMuYDs.png)](https://imgur.com/NuMuYDs.png)

### Commit and push your changes

Once you're done editing the files, go on CMD (Prompt shell or Terminal) and type the following commands one by one:
```shell
git add .
git commit -m "your commit message"
git push
```

### Make a pull request

Go on your forked repository and click on the "Pull requests" tab.

[![Pull Requests](https://imgur.com/RVm7Ulg.png)](https://imgur.com/RVm7Ulg.png)

Click on "New pull request".

[![New Pull Request](https://imgur.com/mbadpiF.png)](https://imgur.com/mbadpiF.png)

Click on "Create pull request", then describe your changes and you've finished.
